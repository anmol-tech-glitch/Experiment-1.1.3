# 📊 Admin Dashboard

A responsive admin dashboard built using HTML, CSS Grid, and JavaScript.
It includes a modern layout with dark/light theme switching and persistent user preferences.

---

## 🚀 Features

* 📐 Grid-based responsive layout (CSS Grid)
* 🌙 Dark / ☀️ Light mode toggle
* 🎨 Theme management using CSS variables
* 💾 Theme preference saved with localStorage
* 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Grid + Variables + Media Queries)
* JavaScript (DOM + localStorage)

## ⚙️ How It Works

* Layout is created using **CSS Grid template areas**
* Theme colors are controlled using **CSS variables**
* JavaScript toggles the theme by adding/removing a class
* Selected theme is saved in **localStorage** and loaded on refresh

---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in any browser
3. Use the toggle button to switch themes
